# Programme 2026 – St-Léger

Static page to display and download the 2026 St-Léger programme in PDF.

## Files

- `index.html` — main page with PDF viewer and download button
- `style.css` — page styles
- `programme.pdf` — the programme PDF (replace with the final document)

## Usage

Host the three files on any static web server or open `index.html` directly in a browser.
To update the programme, replace `programme.pdf` with the new file (keep the same filename).
